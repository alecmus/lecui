\doxysubsection{Detailed Description}
Form base class. The user must inherit from this class and call one of the two constructors from its own constructor. 

Fully DPI aware, but needs to be recreated if the system DPI changes while already open to rescale properly.

{\footnotesize\ttfamily void liblec\+::lecui\+::form\+::reload (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption})}



Reload the form and all its widgets. A high resource reload of the entire form\textquotesingle{}s graphic resources. The following things happen when this method is called\+: 


\begin{DoxyEnumerate}
\item The form\textquotesingle{}s graphic resources are all recreated.
\item The graphic resources for the currently displayed page are all recreated.
\item The graphic resources of all pages that have been added to the form but are not currently displayed are all recreated.
\end{DoxyEnumerate}

Depending on the computing platform, this can take a considerable amount of time in a large app, ranging from milliseconds to actual seconds. The better the app is designed the less time it will take, as such applications typically only create pages on-\/demand. However, even in a well designed app the difference with \mbox{\hyperlink{classliblec_1_1lecui_1_1form_a144e42ae28a7c24d470d5b9b147efebd}{update()}} will be obvious if many pages are currently open. Note\+: this doesn\textquotesingle{}t change the form\textquotesingle{}s \textquotesingle{}state\textquotesingle{}, e.\+g. if a widget was closed it won\textquotesingle{}t magically reappear. This reloads the current \textquotesingle{}state\textquotesingle{}. \mbox{\Hypertarget{classliblec_1_1lecui_1_1form_a65f0164f46c8bd3d61f8733bd0a45271}\label{classliblec_1_1lecui_1_1form_a65f0164f46c8bd3d61f8733bd0a45271}}
