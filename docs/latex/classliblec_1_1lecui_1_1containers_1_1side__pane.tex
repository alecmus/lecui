\doxysubsection{Detailed Description}
Side pane container. To instantiate and add to a container use the \mbox{\hyperlink{classliblec_1_1lecui_1_1containers_1_1side__pane_a5989d8e99da34397a3affad9a27268f5}{add}} static method. 

The side pane is essentially a special left side pane that extends into the title bar. Hence, refer to the status pane documentation remarks for additional information relating to all side panes.

Ensure that there is a maximum of one side pane per form and that there isn\textquotesingle{}t a left status pane.
