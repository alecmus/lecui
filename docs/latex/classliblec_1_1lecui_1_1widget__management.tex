Avoid closing a widget from within its own handler to avoid access violation errors. Also make sure to never keep \textquotesingle{}dangling widget references\textquotesingle{} because attempting to use those references after this method is called will result in access violation errors as well. Make sure to employ proper logic. As a failsafe, the widget is not closed immediately when this method is called. Rather a special timer is scheduled internally to close the widget.
