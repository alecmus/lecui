\doxysubsection{Detailed Description}
Group container. To instantiate and add to a container use the \mbox{\hyperlink{classliblec_1_1lecui_1_1containers_1_1group_aba5db11bc22cd7ead89306934178d851}{add}} static method.

A group does not change the widget path, e.\+g. A widget located at \char`\"{}home\+\_\+page/tab\+\_\+pane/tab three/pane\+\_\+two/label\char`\"{} keeps that location even after being added to a group. Never add a group name to a path. Groups are strictly for aesthetics and they don\textquotesingle{}t \char`\"{}carry\char`\"{} the widgets that belong to them like other containers.
