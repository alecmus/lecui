\doxysubsection{Detailed Description}
Slider widget. To instantiate and add to a container use the \mbox{\hyperlink{classliblec_1_1lecui_1_1widgets_1_1slider_ae00e2b9afeedb591a503fc5ebb59ee5a}{add}} static method.

The slider\textquotesingle{}s orientation is computed automatically based on the its size. If the width is greater than the height it\textquotesingle{}s oriented horizontally, else vertically. With the defaults, the recommended minimum height for horizontal orientation is 35px and the recommended minimum with for vertical orientation is 40px.
