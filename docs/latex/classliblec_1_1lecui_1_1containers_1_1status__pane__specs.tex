\doxysubsection{Detailed Description}
Status pane container specifications.

Note that this does not inherit from \mbox{\hyperlink{classliblec_1_1lecui_1_1widgets_1_1widget}{widgets\+::widget}} as regular panes do. This is by design because of the nature of the status pane.
