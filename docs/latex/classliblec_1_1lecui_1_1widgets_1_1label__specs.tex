\doxysubsection{Detailed Description}
Label widget specifications. 

This widget supports two kinds of formatting\+:


\begin{DoxyEnumerate}
\item HTML formatting with CSS style tags
\end{DoxyEnumerate}

e.\+g. \char`\"{}$<$html$>$\+This is $<$span style=\textquotesingle{}font-\/size\+: 16.\+5pt; font-\/style\+: italic; color\+: 0x0000\+FFFF;\textquotesingle{}$>$good$<$/span$>$.$<$/html$>$\char`\"{};

The supported tags are as follows\+:

strong Bold text em Italic text u Underline text s Strikethrough text span Use CSS style tags

The span tag supports the style attribute and the following properties\+:

font-\/family The font name font-\/size The font size, either in pt or in px font-\/weight The font weight, either \textquotesingle{}none\textquotesingle{} or \textquotesingle{}bold\textquotesingle{} font-\/style The font style, either \textquotesingle{}none\textquotesingle{} or \textquotesingle{}italic\textquotesingle{} text-\/decoration The text decoration, either \textquotesingle{}none\textquotesingle{} or \textquotesingle{}underline\textquotesingle{} or \textquotesingle{}line-\/through\textquotesingle{} color The text color, either in the form rgb(255, 0, 0) or 0x\+FF0000\+FF or \#\+FF0000\+FF

Note\+: the strong, em and u html tags are automatically converted to CSS style tags internally.


\begin{DoxyEnumerate}
\item Custom formatting
\end{DoxyEnumerate}

Start and end of XML formatting is marked by the \textquotesingle{}text\textquotesingle{} tag\+: $<$text$>$$<$/text$>$

The supported attributes of the text are\+:

font The font name size The font size, in points bold \textquotesingle{}true\textquotesingle{} or \textquotesingle{}false\textquotesingle{} italic \textquotesingle{}true\textquotesingle{} or \textquotesingle{}false\textquotesingle{} underline \textquotesingle{}true\textquotesingle{} or \textquotesingle{}false\textquotesingle{} strikethrough \textquotesingle{}true\textquotesingle{} or \textquotesingle{}false\textquotesingle{} color 0x\+AABBCCDD where r = AA, g = BB, b=CC, a=DD

e.\+g. \char`\"{}\+This is $<$text size=\textquotesingle{}16.\+5\textquotesingle{} italic=\textquotesingle{}true\textquotesingle{} color=\textquotesingle{}0x0000\+FFFF\textquotesingle{}$>$good$<$/text$>$.\char`\"{};

Both of the above two methods of formatting will display a label with the sentence\+:

This is good.

The word \textquotesingle{}good\textquotesingle{} will be of size 16.\+5, italic and blue, while the rest of the text will use the widget specifications.

Recommended size with defaults is 200x20px.
