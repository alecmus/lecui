To use the same image and have it appear the same size on screens with different DPI settings set the \mbox{\hyperlink{}{dpi\+\_\+aware}} flag to true. However, if you would like to have only a sharp, pixel-\/perfect splash screen it is recommended to set the flag to false and (optionally) load a different resource based on the dpi scale reported through the \mbox{\hyperlink{classliblec_1_1lecui_1_1form_a045f0d9eed1c1ede57fa9618b49fdbea}{form\+::get\+\_\+dpi\+\_\+scale()}} method.
