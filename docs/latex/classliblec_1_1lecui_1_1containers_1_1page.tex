\doxysubsection{Detailed Description}
Page container. 

This is the generic container type, and is the only one of the containers that can be added directly to a form. Any widget and any container can be added to it. It comes full featured with scroll bars that appear if anything within it exceeds it\textquotesingle{}s dimensions. It covers the entire form save for the border and the titlebar. It\textquotesingle{}s size is tightly coupled to the form, and is resized automatically as the form is resized. As such, it\textquotesingle{}s size cannot be directly manipulated. Other container types, however, can be resized directly, e.\+g. panes, but then they cannot be added directly to a form.
