overridevirtual

Check whether a point is within the widget.

Parameters
pointThe point, in pixels.
Returns
True if point is within the widget, false otherwise.

For in-widget hit-testing avoid overriding this virtual function directly; rather use the protected overload. Also remember to factor in the dpi scale otherwise hit testing will not work properly in high-dpi environments.
