Parameters
fmA reference to the form.
guidUnique guid used to prevent multiple instances of the app, e.g. "Global\\{7DBACCA2-B96E-4D0E-B0F7-933EA9A3C064}". This same guid is also used to allow a new instance to (1) Cause a new instance to find and open the existing instance before exiting (2) Pass it's command line arguments as data to the existing instance. The latter is received in the existing instance through the form's on_receive_data handler.

It is highly recommended that top level forms within the same app use the same guid. This way, when a new instance is attempted, the latest form to be created will be brought to the foreground. GUIDs only apply to top level forms.
