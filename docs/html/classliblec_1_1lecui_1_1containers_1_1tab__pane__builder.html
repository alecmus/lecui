Detailed Description

Tab pane container builder.

Only tab containers can be added to the tab pane container.

Select the visible tab in the tab pane.

Parameters
tab_nameThe in-control unique name of the tab, e.g. "Options".

By default none of the tabs is selected if this method is never called and the tab pane will load empty. The first selection will happen when the user clicks a tab, at which point the contents of that tab will be displayed.
