Detailed Description

Tab container builder.

Any widget can be added to this container. Consequently, recursion is fully supported, allowing tab panes within tabs that are themselves in another tab pane, to virtually any depth level that the memory of the computer the app is running on can permit.

Get the tab container page.

Returns
A reference to the tab container page.

Note that this is a container of type 'page', hence anything that can be added to a regular page can be added here as well. The page comes fully featured with scroll bars when widgets exceed the dimensions of the tab, just like a regular page.
